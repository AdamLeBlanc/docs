If you import your project from a GitHub or Bitbucket repo, Forestry will commit content updates back to that repo. It will also watch that repo for changes and pull them in so you can continue to push code to it.



When an editor saves changes, 2 things happen:

  1. All changes are committed and pushed to your GitHub repo
  2. The site is compiled and sent to your host (FTP, S3, GitHub pages, etc)
