If you want default fields for new posts, you can define them in your _config.yml file (see the Jekyll docs for more info). In this example below, all new posts (no matter what the path is).

The following example will generate a field and an field for all new posts:

<br> <pre><br> defaults:<br> -<br> scope:<br> path: ""<br> type: posts<br> values:<br> work: true<br> Author: "Scott Gallant"<br> </pre><br>
