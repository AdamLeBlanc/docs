## setup
- from the `docs` directory
- `npm install -g gulp`
- `npm install -g node-sass`
- `npm install`
- 'git clone git@github.com:forestryio/forestry-doc-theme.git hugo/themes/forestry-doc-theme' to the `./themes/` directory

## gulp commands

- `gulp scss`
  - Build scss, and generate source maps
